Background

This is an API (obviously) that I wrote to interact with my Raspberry Pi 3 B. It currently just exposes various stats about the system. I hope to add more information in the future. I also hope to add other fun functionality in the future.

This project was initally built for a summer web development boot camp that I attended in the summer of 2018. Since then it has been greatly expanded upon (You can just check the commit history for proof of that!). Initally the the Raspberry Pi also served the website itself. But now the Pi has just been converted into a resource API (and docs repository), while site is served somewhere else.

Technologies Used

  • Flask Micro-Framework
  • SQL-Alchemy ORM
  • SQLite
  • Nginx
